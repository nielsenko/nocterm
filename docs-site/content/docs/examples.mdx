The `example/` directory contains 66 examples demonstrating nocterm's features. Run any example with:

```bash
dart run example/example_name.dart
```

## Featured Examples

These showcase nocterm's full potential:

| Example | Description |
|---------|-------------|
| `task_manager_demo.dart` | Full system monitor with dashboard, processes, performance graphs, network connections |
| `chat_demo.dart` | Complete chat interface with auto-scroll, message history, and input |
| `theme_showcase.dart` | Interactive demo of all 6 built-in themes with live switching |
| `showcase_demo.dart` | Feature showcase demonstrating core capabilities |
